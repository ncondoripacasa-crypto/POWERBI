# ExcelPro - Aprendizaje Gamificado de Excel

ExcelPro es una plataforma web interactiva y gamificada diseñada para enseñar Microsoft Excel desde cero. Ideal para estudiantes universitarios, de secundaria o cualquier persona que desee dominar esta herramienta a través de la práctica directa.

## 🚀 Características
- **Gamificación Integrada:** Sistema de experiencia (XP), niveles desbloqueables e insignias.
- **Aprendizaje Interactivo:** Lecciones teóricas seguidas de ejercicios prácticos de validación inmediata.
- **Persistencia de Datos:** El progreso del estudiante se guarda localmente en su navegador usando \`localStorage\`.
- **Sin Frameworks:** Construido exclusivamente con HTML5, CSS3 y JavaScript moderno (Vanilla).
- **Diseño Responsivo:** Interfaz adaptada tanto para computadoras de escritorio como para dispositivos móviles.

## 📁 Estructura del Proyecto
- \`index.html\`: Archivo principal que contiene la Landing Page y las vistas dinámicas de los niveles.
- \`styles.css\`: Hoja de estilos con variables CSS, animaciones y diseño responsivo.
- \`script.js\`: Lógica de la aplicación, control de estado, validación de respuestas y renderizado dinámico.
- \`assets/\`: Carpeta preparada para futuras imágenes o recursos gráficos estáticos.

## 🎓 Contenido Educativo
El curso consta de 5 niveles progresivos:
1. **Introducción a Excel:** Conceptos de columnas, filas y celdas.
2. **Operaciones Básicas:** Uso de operadores aritméticos (+, -, *, /).
3. **Funciones Básicas:** \`SUMA\`, \`PROMEDIO\`, \`MAX\`, \`MIN\`.
4. **Funciones de Texto:** \`CONCATENAR\`, \`IZQUIERDA\`, \`DERECHA\`.
5. **Funciones de Fecha:** \`HOY\`, \`AHORA\`, \`AÑO\`.

## ⚙️ Cómo ejecutar localmente
1. Clona o descarga este repositorio en tu computadora.
2. Navega a la carpeta del proyecto.
3. Haz doble clic en el archivo \`index.html\` para abrirlo en tu navegador predeterminado (Chrome, Firefox, Edge, etc.).
4. ¡Comienza a aprender!

## 🌐 Cómo desplegar en GitHub Pages
Para que el proyecto sea accesible a cualquier persona por internet:
1. Crea un repositorio en [GitHub](https://github.com/).
2. Sube los archivos de este proyecto al repositorio.
3. Ve a la pestaña **Settings** (Configuración) de tu repositorio.
4. En el menú lateral izquierdo, haz clic en **Pages** (Páginas).
5. En la sección *Source* (Fuente), selecciona la rama \`main\` o \`master\` y haz clic en **Save** (Guardar).
6. GitHub te proporcionará un enlace donde tu proyecto estará activo en unos minutos (ej. \`https://tu-usuario.github.io/tu-repositorio/\`).
